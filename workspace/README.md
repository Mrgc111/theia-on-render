# 🎯 Theia IDE Workspace

このディレクトリは開発作業領域です。

## 🚀 クイックスタート

### ファイル操作
- **新規ファイル**: 右クリック → New File
- **新規フォルダ**: 右クリック → New Folder
- **ファイル検索**: `Ctrl + P`

### ターミナル
- **ターミナル起動**: `Ctrl + Shift + ` `
- **新しいターミナル**: `Ctrl + Shift + 5`
