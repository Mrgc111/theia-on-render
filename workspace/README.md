# Workspace

このフォルダでプロジェクトを作成してください。

## 使い方

1. Theia IDEが起動したら、このworkspaceフォルダが自動的に開かれます
2. ここに新しいプロジェクトフォルダを作成してください
3. 例：
   - my-web-app/
   - sample-project/
   - etc.

## 例

```bash
mkdir my-project
cd my-project
npm init -y
```

## Theia IDEの基本操作

- **ファイル作成**: 左サイドバーで右クリック → New File
- **ターミナル**: `Ctrl+Shift+`` (バッククォート)
- **コマンドパレット**: `F1` または `Ctrl+Shift+P`
- **ファイル検索**: `Ctrl+P`
- **全体検索**: `Ctrl+Shift+F`
